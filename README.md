# Movie Library

**Автор:** Попов Роман

## Описание
Приложение для хранения и фильтрации личной кинотеки. Позволяет добавлять фильмы с указанием названия, жанра, года выпуска и рейтинга. Данные сохраняются в JSON-файл. Реализована фильтрация по жанру и году.

## Возможности
- Добавление фильмов с проверкой ввода.
- Фильтрация по жанру и году.
- Сохранение и загрузка данных из JSON.
- Простой графический интерфейс на Tkinter.

## Примеры использования
1. Запустите `movie_library.py`.
2. Введите данные о фильме и нажмите «Добавить фильм».
3. Используйте фильтры для поиска нужных фильмов.
4. Сохраните данные кнопкой «Сохранить», загрузите — «Загрузить».
